Tất cả các thẻ game đều chứa một phiên bản Phần mềm Hệ thống phù hợp để khởi chạy game, giúp cho người dùng phiên bản thấp hơn nâng cấp lên phiên bản phù hợp (kể cả khi không có mạng Internet).
{: .notice--info}

Cập nhật phần mềm sử dụng thẻ game chỉ nâng cấp những phần cốt lõi của hệ thống, như phần Tuỳ chỉnh Hệ thống, Màn hình Home Menu, phần mềm Nintendo 3DS Sound, v.. v. Cập nhật bằng thẻ sẽ không nâng cấp phần tính năng liên quan tới Mạng, như công cụ Chuyển đổi hệ thống (System Transfer), trình duyệt, StreetPass Mii Plaza hoặc eShop.
{: .notice--info}

This means that a cart update will NOT install a browser if you do not already have one. This will **only** update the actual system version *(for example, 2.1.0-0 -> 6.0.0-0)*. The last number *(the -0)* will **not** change.
{: .notice--warning}

**Updating with a game cart containing the version 9.9.0 or above will remove your browser!**
{: .notice--danger}

#### What you need

* A game cart that [contains the version](http://www.3dsdb.com/) specified in the chart

#### Instructions

1. Boot your 3DS into recovery mode by holding L+R+A+UP while powering on
2. Decline the update and reboot
3. Put the game in the system
4. Attempt to launch the game
5. Update when prompted
