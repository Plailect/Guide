Essa é uma seção complementar para mover o conteúdo de um EmuNAND prévio para o seu novo CFW de SysNAND, e então remover a antiga partição de EmuNAND. Lembre que os termos EmuNAND e RedNAND são implementações ligeiramente diferentes do [mesmo conceito](http://3dbrew.org/wiki/NAND_Redirection).
{: .notice--info}

Você vai PERDER seus saves de VC de GBA e DSiWare!
{: .notice--danger}

**Você DEVE já ter concluído a instalação do arm9loaderhax + Luma3DS para usar isso**
{: .notice--danger}

#### O que você precisará

* Uma EmuNAND existente
* A versão mais recente do [GodMode9](https://github.com/d0k3/GodMode9/releases/latest)

#### Instruções

1. Copie `GodMode9.bin` do GodMode9 `.zip` para a pasta `/luma/payloads/` no seu cartão SD e renomeie `GodMode9.bin` no `/luma/payloads/` para `up_GodMode9.bin`
2. Reinsira o cartão SD no seu 3DS
3. Abra o Hourglass9 pelo arm9loaderhax segurando (Start) ao ligar
4. Vá para "EmuNAND Backup/Restore" e, em seguida, selecione a opção "EmuNAND Backup" para fazer backup da sua EmuNAND para 'NANDmin_emu.bin'
5. Pressione (B) para voltar para o menu principal
6. Vá para "SysNAND Backup/Restore" e, em seguida, selecione a opção "SysNAND Restore (keep a9lh)" para restaurar da SysNAND de 'NANDmin_emu.bin'
7. Aperte (Select) para ejetar o cartão SD
8. Coloque o seu cartão SD no seu computador, então copie `NANDmin.bin` e `NANDmin.bin.sha` da pasta `/files9/` no seu cartão SD para um local seguro
  + Faça backups em vários locais
  + Este backup vai salvá-lo de um brick caso algo dê errado no futuro
  + **O tamanho do seu backup deve ser o igual a um dos contidos em [this](nand-size) página; senão, você deve apagá-lo e fazer um novo!**
9. Exclua 'NANDmin_emu.bin' da pasta '/ files9 /' do cartão SD depois de copiá-lo
10. * * Faça backup de tudo do cartão SD para uma pasta no seu computador, todos os arquivos serão apagados no próximo passo * *
11. Reinsira o cartão SD no seu 3DS
12. Pressione (Start) para reiniciar, mantendo o (cima) apertado para abrir o GodMode9
13. Pressione o botão (Home) para abrir o menu de ação
14. Selecione "SD format menu"
15. Pressione (A) para confirmar
16. Selecione "No EmuNAND"
17. Selecione "Auto"
18. Digite a combinação de botões solicitada para confirmar
19. Segure (R) e pressione (B) ao mesmo tempo para ejetar o cartão SD
20. Coloque o seu cartão SD no seu computador e, em seguida, copie todos os arquivos de volta para ele
  + Certifique-se de que substituir o arquivo 'arm9loaderhax.bin' no cartão SD com o do seu backup
21. Reinsira o cartão SD no seu 3DS
22. Pressione (A) para remontar seu cartão SD
23. Pressione (Start) para reiniciar!
24. Se você receber uma tela preta, [siga este guia] (troubleshooting#ts_sys_down) de solução de problemas

Retornar para [Instalando arm9loaderhax](installing-arm9loaderhax).
