<a name="faq_latestfw" />**P:** *Estou na última versão do sistema, meu console é hackeável?*    
**R:**Yes, 11.2 é hackeável usando o método do [Homebrew Launcher (Soundhax)] (https://3ds.guide/homebrew-launcher-(soundhax))

<a name="faq_updatecfw" />**P:** *Como eu atualizo a Luma3DS?*    
**R:** Use o Luma3DS Updater

<a name="faq_gatewaysky" />**P:** *Eu ouvi falar sobre essa coisa que devo comprar (Gateway, Sky3DS, etc), é algo que eu preciso?*    
**R:** Não. O arm9loaderhax é um método superior de carregar a hax que roda bem cedo na inicialização do sistema e abre a oportunidade para coisas como uma verdadeira firmware customizada.

<a name="faq_gatewaysaves" />**P:** *Como eu transfiro saves de uma instalação Gateway para os jogos instalados no sistema?*    
**R:** Veja [isto] (https://gbatemp.net/threads/425743/)

<a name="faq_need" />**P:** *Do que eu preciso para este guia?*    
**R:** Este guia pode ser completado com programas gratuitos e (se você não possuir o navegador) um jogo compatível   

<a name="faq_risky" />**P:** *O quão arriscado é hackear meu console?*    
**R:** Bricks no momento são *basicamente* impossíveis a menos que você ignore/pule as verificações de segurança.

<a name="faq_homebrew" />**P:** *Posso rodar os incríveis homebrews e emuladores com isso?*    
**R:** Sim! Custom Firmware permite não somente o Homebrew Launcher, mas também lhe dá acesso aos hacks em todas as versões, o que significa que você o mantém, mesmo que atualize.

<a name="faq_regionfree" />**P:** *Posso usar isso para jogar jogos de outras regiões?*    
**R:** Luma3DS suporta patches de Region Free, que permitem aos usuários instalar jogos de qualquer região sem modificações ou passos extras.

<a name="faq_updates" />**P:** *É seguro atualizar a SysNAND com CFW?*    
**R:** Sim. Isso é seguro, pare de perguntar.

<a name="faq_support" />**Q:** *Where should I go for support?*    
**A:** For support, ask for help at [#3dshacks on Rizon IRC](https://www.reddit.com/r/3dshacks/wiki/irc) or [3DS Hacking on Discord](https://discord.gg/MWxPgEp).  

<a name="faq_le4gbsd" />**Q:** *Can I do this with a 4GB (or smaller) SD card?*    
**A:** Thanks to the new CTRNAND method, yes!

<a name="faq_ge128gbsd" />**Q:** *Can I use a 128 GB (or larger) SD card?*    
**A:** Yes, but some users have reported slowdowns when using such large sizes. Note that you cannot format the card with exFAT; you need to format it with FAT32.

<a name="faq_movesd" />**Q:** *How do I move to a new SD card at the end of this?*    
**A:** Just copy all the files to the new SD card.

<a name="faq_NNID" />**Q:** *Can I keep my NNID?*    
**A:** If you start with an NNID and follow every step, you will end up keeping your NNID at the end.

<a name="faq_systransfer" />**Q:** *What about System Transfers?*    
**A:** Once you have completed the guide, you can system transfer to and from an arm9loaderhax'd 3DS or 2DS exactly like you would a regular stock console or EmuNAND (you will lose any non-legitimate games/themes/DLC in the transfer; saves will stay even for games that are lost).

<a name="faq_nopc" />**Q:** *Can I do this without a computer (e.g. an Android phone)?*    
**A:** All you need is the ability to put files on an SD card!

<a name="faq_problem" />**Q:** *Help! Something bad happened and now I cannot boot...*    
**A:** Please look at the [troubleshooting guide](troubleshooting).
