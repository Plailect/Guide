Этот дополнительный раздел предназначен тем, кто хочет перенести содержимое своего EmuNAND в новый SysNAND с кастомной прошивкой. В процессе мы так же удалим раздел с EmuNAND с карты памяти, что освободит немного места на карте памяти. Помните, что RedNAND и EmuNAND - слегка разные реализации одного и того же [концепта](http://3dbrew.org/wiki/NAND_Redirection).
{: .notice--info}

Примечания: по окончанию, вы лишитесь ВСЕХ своих сохранений в GBA VC и DSiWare играх!
{: .notice--danger}

**На момент начала работы, у вас УЖЕ должна быть установлена связка arm9loaderhax + Luma3DS**
{: .notice--danger}

#### Требуется

* Существующая EmuNAND
* Самый свежая версия [GodMode9](https://github.com/d0k3/GodMode9/releases/latest)

#### Инструкция

1. Скопируйте `GodMode9.bin` из GodMode9 `.zip` архива в директорию `/luma/payloads/` на карте памяти, и в ней переименуйте `GodMode9.bin` в `up_GodMode9.bin`
2. Вставьте SD карту обратно в приставку
3. Запустите Hourglass9, зажав и держа кнопку (Start) при включении консоли
4. Перейдите в пункт "EmuNAND Backup/Restore", где выберите опцию "EmuNAND Backup", чтобы сделать резервную копию вашего EmuNAND под именем файла `NANDmin_emu.bin`
5. Нажмите (B) для возврата в главное меню программы
6. Перейдите в пункт "SysNAND Backup/Restore", где затем выберите опцию "SysNAND Restore (keep a9lh)" для восстановления вашего SysNAND из файла `NANDmin_emu.bin`
7. Нажмите (Select) для режима безопасного извлечения карты и достаньте SD карту из консоли
8. Подключите карту памяти к компьютеру и затем скопируйте файлы `NANDmin_emu.bin` и `NANDmin_emu.bin.sha` из `/files9/` директории на карте памяти куда-нибудь на компьютер в безопасное место
  + В целях безопасности, сохраните копии этих файлов на нескольких устройствах и/или вашем хранилище в интернете
  + This backup will save you from a brick if anything goes wrong in the future
  + **Your backup should match one of the sizes on [this](nand-size) page; if it does not, you should delete it and make a new one!**
9. Удалите `NANDmin_emu.bin` и `NANDmin_emu.bin.sha` из папки `/files9/` после копирования
10. **Сделайте резервные копии всех файлов на SD карте, поскольку в следующем шаге карта будет отформатирована.**
11. Верните SD карту в приставку
12. Нажмите (Start) для перезагрузки и удерживайте (Вверх), чтобы запустить GodMode9
13. Нажмите кнопку (Home) для вызова меню
14. Выберите "SD format menu"
15. Нажмите (A) для подтверждения
16. Выберите "No EmuNAND"
17. Выберите "Auto"
18. Введите указанную последовательность кнопок для подтверждения
19. Удерживая (R) нажмите (B) для того, чтобы извлечь карту памяти
16. Вставьте карту памяти в компьютер и скопируйте на нее файлы, которые вы сохранили несколько шагов назад
  + Убедитесь, что скопировали с заменой файл `arm9loaderhax.bin`. То есть на карте памяти должен быть файл, который вы копировали на компьютер, вместе со всем содержимым карты памяти до ее форматирования.
18. Вставьте SD карту в приставку
19. Нажмите (A) чтобы смонтировать карту памяти
20. Нажмите (Start) для перезагрузки!
19. Если нарветесь на черный экран, перейдите к разделу с [проблемами и их решениями](troubleshooting#ts_sys_down)

Вернитесь в раздел [установку arm9loaderhax](installing-arm9loaderhax).
