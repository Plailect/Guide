<a name="faq_latestfw" />**Q:** *Моя консоль на последней версии прошивки, могу ли я её взломать?*    
**A:** Да, версия 11.2 может быть взломана с помощью [Homebrew Launcher (SoundHax)](https://3ds.guide/homebrew-launcher-(soundhax))

<a name="faq_updatecfw" />**Q:** *Как можно обновить Luma3DS?*    
**A:** Используйте Luma3DS Updater

<a name="faq_gatewaysky" />**Q:** *I heard about this thing I have to pay for (Gateway, Sky3DS, etc), is that something I need?*    
**A:** Нет. arm9loaderhax - это прогрессивный метод взлома, который запускается в ранних стадиях загрузки консоли и открывает возможность для таких вещей, как полностью кастомные прошивки.

<a name="faq_gatewaysaves" />**Q:** *Как я могу перенести свои сохранения с карты Gateway на саму консоль для уже установленных игр?*    
**A:** Ознакомьтесь с [этим ресурсом](https://gbatemp.net/threads/425743/) (английский)

<a name="faq_need" />**Q:** *What do I need to do this guide?*    
**A:** This guide can be completely done with free software and (if your device has no browser) a compatible game   

<a name="faq_risky" />**Q:** *Насколько опасно взламывать консоль?*    
**A:** На данный момент практически невозможно испортить консоль, если не игнорировать все меры предосторожности.

<a name="faq_homebrew" />**Q:** *Смогу ли я после этого запускать сторонние программы и эмуляторы?*    
**A:** Конечно! Custom Firmware not only enables the Homebrew Launcher, but it also will give you hacks on all versions, which means you keep it forever, even with updates.

<a name="faq_regionfree" />**Q:** *Смогу ли я играть в игры других регионов?*    
**A:** Luma3DS supports Region Free patches, which enables users to install games from any region without any extra modification or steps.

<a name="faq_updates" />**Q:** *Is it safe to update CFW SysNAND?*    
**A:** Да. Это безопасно, хватит уже спрашивать об этом.

<a name="faq_support" />**Q:** *Куда я могу обратиться за поддержкой?*    
**A:** Если у вас есть вопросы, вы можете задать их на [#3dshacks канале Rizon IRC](https://www.reddit.com/r/3dshacks/wiki/irc) или [3DS Hacking канале в Discord](https://discord.gg/MWxPgEp) (английский).  

<a name="faq_le4gbsd" />**Q:** *Могу ли я использовать SD-карты размером 4 ГБ (или меньше)?*    
**A:** Да, благодаря новому CTRNAND методу.

<a name="faq_ge128gbsd" />**Q:** *Могу ли я использовать SD-карты размером 128 ГБ (или больше)?*    
**A:** Yes, but some users have reported slowdowns when using such large sizes. Стоит уточнить, что нельзя форматировать карту в exFAT; необходимо использовать FAT32.

<a name="faq_movesd" />**Q:** *How do I move to a new SD card at the end of this?*    
**A:** Просто скопируйте все файлы на новую SD-карту.

<a name="faq_NNID" />**Q:** *Могу ли я сохранить свой NNID?*    
**A:** If you start with an NNID and follow every step, you will end up keeping your NNID at the end.

<a name="faq_systransfer" />**Q:** *What about System Transfers?*    
**A:** Once you have completed the guide, you can system transfer to and from an arm9loaderhax'd 3DS or 2DS exactly like you would a regular stock console or EmuNAND (you will lose any non-legitimate games/themes/DLC in the transfer; saves will stay even for games that are lost).

<a name="faq_nopc" />**Q:** *Can I do this without a computer (e.g. an Android phone)?*    
**A:** Все, что вам понадобится, это возможность копировать файлы на SD-карту!

<a name="faq_problem" />**Q:** *Help! Что-то пошло не так, и консоль больше не запускается...*    
**A:** Please look at the [troubleshooting guide](troubleshooting).
