<a name="faq_latestfw" />**Q:** *Моя консоль на последней версии прошивки, могу ли я её взломать?*    
**A:** Да, версия 11.2 может быть взломана с помощью [Homebrew Launcher (SoundHax)](https://3ds.guide/homebrew-launcher-(soundhax))

<a name="faq_updatecfw" />**Q:** *Как можно обновить Luma3DS?*    
**A:** Следует использовать Luma3DS Updater

<a name="faq_gatewaysky" />**Q:** *Я слышал, что в продаже есть такое устройство (Gateway, Sky3DS, и т. д.), мне это необходимо?*    
**A:** Нет. arm9loaderhax - это прогрессивный метод взлома, который запускается в ранних стадиях загрузки консоли и открывает возможность для таких вещей, как полностью кастомные прошивки.

<a name="faq_gatewaysaves" />**Q:** *Как я могу перенести свои игровые сейвы с Gateway карты на саму консоль для уже установленных игр?*    
**A:** Ознакомтесь с [этим ресурсом](https://gbatemp.net/threads/425743/) (английский)

<a name="faq_need" />**Q:** *Что мне потребуется в ходе использования этого гайда?*    
**A:** Все, что вам понадобится, это бесплатные программы и (если у вас нет возможности использовать эксплоит для браузера) подходящая игра.   

<a name="faq_risky" />**Q:** *Есть ли риски испортить консоль в процессе взлома?*    
**A:** На данный момент порча консоли практически невозможна, если не игнорировать все приведенные указания.

<a name="faq_homebrew" />**Q:** *Смогу ли я после этого запускать сторонние программы и эмуляторы?*    
**A:** Конечно! Кастомная прошивка не только поддерживает Homebrew Launcher, но и также остаётся на консоли даже после обновлений.

<a name="faq_regionfree" />**Q:** *Смогу ли я играть в игры других регионов?*    
**A:** Luma3DS поддерживает Region Free патчи, которые позволяют пользователям запускать игры любого региона без лишних телодвижений.

<a name="faq_updates" />**Q:** *Безопасно ли обновлять SysNAND прошивку на уже взломанной консоли?*    
**A:** Да. Это безопасно, хватит уже спрашивать об этом.

<a name="faq_support" />**Q:** *Куда я могу обратиться за поддержкой?*    
**A:** Если у вас есть вопросы, вы можете задать их на [#3dshacks канале Rizon IRC](https://www.reddit.com/r/3dshacks/wiki/irc) или [3DS Hacking канале в Discord](https://discord.gg/MWxPgEp) (английский).  

<a name="faq_le4gbsd" />**Q:** *Хватит ли мне SD карты на 4 ГБ или меньше?*    
**A:** Да, благодаря новому CTRNAND методу.

<a name="faq_ge128gbsd" />**Q:** *Могу ли я использовать SD карты на 128 ГБ и больше?*    
**A:** Да, но некоторые пользователи сообщали о замедлениях в работе при использовании таких больших накопителей. Стоит уточнить, что нельзя форматировать карту с exFAT; необходимо использовать FAT32.

<a name="faq_movesd" />**Q:** *Как мне перенести данные на новую SD карту?*    
**A:** Просто скопируйте все файлы на новую карту памяти.

<a name="faq_NNID" />**Q:** *Могу ли я избежать потери своего NNID?*    
**A:** Если точно следовать этому гайду от начала до конца, по окончанию установки у вас будет тот же NNID, что и на момент начала работы.

<a name="faq_systransfer" />**Q:** *А как обстоит с System Transfer?*    
**A:** После прохождения гайда, передача данных возможна в обе стороны для консоли с arm9loaderhax так же, как и для немодифицированной консоли или для консоли с EmuNAND (все нелегальные игры/темы/загружаемый контент будет потерян при передаче, однако сохранения останутся даже для потерянных игр).

<a name="faq_nopc" />**Q:** *Могу ли я обойтись без использования компьютера (например, используя Android телефон)?*    
**A:** Все, что вам понадобится, это возможность копировать файлы на SD карту.

<a name="faq_problem" />**Q:** *Что делать?! Что-то пошло не так, и консоль больше не запускается...*    
**A:** Ознакомьтесь с [перечнем проблем и решений](troubleshooting).
