#### What you need

* The latest release of [Decrypt9WIP](https://github.com/d0k3/Decrypt9WIP/releases/latest/)
* The latest release of [safehax](https://github.com/TiniVi/safehax/releases/latest)
* The latest release of [fasthax](https://github.com/nedwill/fasthax/releases/latest)

#### Instructions

2. Créer un dossier nommé "files9" sur la racine de votre carte SD si celui-ci n'existe pas déjà
3. Copier "safehax.3dsx" dans le dossier "/3ds/" sur votre carte SD
4. Copier "fasthax.3dsx" dans le dossier "/3ds/" sur votre carte SD
3. Copier le fichier "Decrypt9WIP.bin" contenu dans l'archive Decrypt9WIP dans la racine de votre carte SD. Renommer "Decrypt9WIP.bin" en "safehaxpayload.bin"
3. Reinsert your SD card into your 3DS
4. Enter the homebrew launcher
4. Launch fasthax
4. Once it has completed, press (Start) to exit back to the homebrew launcher
  + This may take several tries
5. Launch safehax
6. If the exploit was successful, you will have booted into Decrypt9
