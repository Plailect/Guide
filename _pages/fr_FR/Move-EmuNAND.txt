Ceci est une section ajoutée pour déplacer le contenu d'une EmuNAND précédente sur votre nouveau CFW SysNAND, pour ensuite supprimer l'ancienne partition EmuNAND. Notez que les termes EmuNAND et RedNAND font référence à des implémentations légèrement différentes du [même concept](http://3dbrew.org/wiki/NAND_Redirection).
{: .notice--info}

Vous allez PERDRE vos sauvegardes GBA VC et DSiWare!
{: .notice--danger}

**Vous DEVEZ avoir déjà installé arm9loaderhax + Luma3DS afin d'utiliser cela**
{: .notice--danger}

### Ce qu'il vous faut

* Une EmuNAND existante
* La dernière version de [GodMode9](https://github.com/d0k3/GodMode9/releases/latest)

### Instructions

1. Copier `GodMode9.bin` depuis GodMode9 `.zip` vers le dossier `/luma/payloads/` sur votre carte SD et renomez `GodMode9.bin` dans `/luma/payloads/` en `up_GodMode9.bin`
2. Ré-insérez votre carte SD dans votre 3DS
3. Ouvrez Hourglass9 depuis arm9loaderhax en maintenant (Start) au démarrage
4. Aller sur "EmuNAND Backup/Restore", puis sélectionnez l'option "EmuNAND Backup" de façon à sauvegarder votre EmuNAND dans `NANDmin_emu.bin`
5. Appuyez sur (B) pour retourner sur le menu principal
6. Aller sur "SysNAND Backup/Restore", puis sélectionnez l'option "SysNAND Restore (keep a9lh)" pour restaurer votre SysNand depuis `NANDmin_emu.bin`
7. Appuyez sur (Select) pour éjecter votre carte SD
8. Insérez votre carte SD dans votre ordinateur, puis copiez `NANDmin_emu.bin` et `NANDmin_emu.bin.sha` depuis le dossier`/files9/` sur votre carte SD dans une destination sécurisée
  + Faites de multiples sauvegardes
  + Cette sauvegarde vous sauvera d'un brick si quoi que ce soit de mauvais arrive dans le future
  + **Votre sauvegarde devrait correspondre à une des taille de cette [this](nand-size) page; si cela n'est pas le cas, vous devrez supprimer votre sauvegarde et en refaire une nouvelle!**
9. Supprimez `NANDmin_emu.bin` depuis le dossier `/files9/` sur votre carte SD après l'avoir copié
10. **Sauvegardez tout les fichiers sur votre carte SD dans un dossier sur votre ordinateur, tout les fichiers seront supprimés à l'étape suivante**
11. Ré-insérez votre carte SD dans votre 3DS
12. Appuyez sur (Start) pour redémarrer tout en tenant (Haut) pour ouvrir GodMode9
13. Appuyez sur le bouton (Home) pour afficher le menu
14. Sélectionnez "SD format menu"
15. Appuyez sur (A) pour confirmer
16. Sélectionnez "No EmuNAND"
17. Sélectionnez "Auto"
18. Entrez la combinaison de boutons affichée
19. Maintenez (R) et appuyez sur (B) en même temps pour éjecter votre carte SD
20. Insérez votre carte SD dans votre ordinateur, ensuite copiez dessus tout vos fichiers sauvegardés
  + Assurez vous de remplacer le fichier `arm9loaderhax.bin` avec celui de votre sauvegarde
21. Ré-insérez votre carte SD dans votre 3DS
22. Appuyez sur (A) pour remonter votre carte SD
23. Appuyez sur (Start) pour redémarrer!
24. Si vous avez un écran noir, [suivez ce guide de dépannage](troubleshooting#ts_sys_down)

Retour à [Installer arm9loaderhax](installing-arm9loaderhax).
