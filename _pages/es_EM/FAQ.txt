<a name="faq_latestfw" /> ** P: ** *Estoy en la última versión del sistema, ¿puedo hackear mi dispositivo?*    
**R:** Sí, la versión 11.2 es hackeable usando el metodo [Homebrew Launcher (SoundHax)](https://3ds.guide/homebrew-launcher-(soundhax))

<a name="faq_updatecfw" /> **P:** *¿Como hago para actualizar Luma3DS?*    
**R:** Usa Luma3DS Updater

<a name="faq_gatewaysky" />**P:** *He escuchado de esta cosa que tengo que pagar (Gateway, Sky3DS, etc), es eso algo que necesito?*    
** R: ** No. arm9loaderhax es un mejor metodo para cargar hax que corre al momento en que enciendes la consola y abre la oportunidad para cosas como "custom firmware".

<a name="faq_gatewaysaves" />**P:** *Como muevo datos de guardado de un cartucho Gateway a los juegos instalados en el sistema?*    
**R:** Ve [esto](https://gbatemp.net/threads/425743/)

<a name="faq_need" />**P:** *Qué necesito para hacer esta guía?*    
**R:** Esta guía se puede hacer con software gratis y (si no tienes el navegador) un juego compatible   

<a name="faq_risky" />**P:** *Hay riesgo cuando hackeas la consola?*    
**R:** Los "bricks" son ahora *basicamente* imposibles a menos que ignores/eludas todas los controles de seguridad.

<a name="faq_homebrew" />**P:** *Puedo correr asombrosas aplicaciones homebrew y emuladores con esto?*    
**R:** Yes! El "Custom Firmware" no solo da acceso al Homebrew Launcher, pero también te dará hacks en todas las versiones, lo que significa que lo tendrás para siempre, hasta con actualizaciones.

<a name="faq_regionfree" />**P:** *Puedo usar esto para correr juegos de otras regiones?*    
**R:** Luma3DS soporta parches "Region Free", que dejan al usuario instalar juegos de cualquier región sin modificaciones extras u pasos.

<a name="faq_updates" />**P:** *Es seguro actualizar el CFW de la SysNAND?*    
**R:** Sí. Esto es seguro, dejen de preguntarlo.

<a name="faq_support" />**P:** *Donde debería ir en caso de ayuda?*    
**A:** For support, ask for help at [#3dshacks on Rizon IRC](https://www.reddit.com/r/3dshacks/wiki/irc) or [3DS Hacking on Discord](https://discord.gg/MWxPgEp).  

<a name="faq_le4gbsd" />**Q:** *Can I do this with a 4GB (or smaller) SD card?*    
**A:** Thanks to the new CTRNAND method, yes!

<a name="faq_ge128gbsd" />**Q:** *Can I use a 128 GB (or larger) SD card?*    
**A:** Yes, but some users have reported slowdowns when using such large sizes. Note that you cannot format the card with exFAT; you need to format it with FAT32.

<a name="faq_movesd" />**Q:** *How do I move to a new SD card at the end of this?*    
**A:** Just copy all the files to the new SD card.

<a name="faq_NNID" />**Q:** *Can I keep my NNID?*    
**A:** If you start with an NNID and follow every step, you will end up keeping your NNID at the end.

<a name="faq_systransfer" />**Q:** *What about System Transfers?*    
**A:** Once you have completed the guide, you can system transfer to and from an arm9loaderhax'd 3DS or 2DS exactly like you would a regular stock console or EmuNAND (you will lose any non-legitimate games/themes/DLC in the transfer; saves will stay even for games that are lost).

<a name="faq_nopc" />**Q:** *Can I do this without a computer (e.g. an Android phone)?*    
**A:** All you need is the ability to put files on an SD card!

<a name="faq_problem" />**Q:** *Help! Something bad happened and now I cannot boot...*    
**A:** Please look at the [troubleshooting guide](troubleshooting).
