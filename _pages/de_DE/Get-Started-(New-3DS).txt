Wählen Sie die entsprechende Seite für Ihre Version aus der unten folgenden Tabelle.
{: .notice--primary}

{% capture notice-1 %}
Die ersten beiden Spalten beziehen sich auf die letzte Nummer Ihrer Version (welche zeigt, welche Version des Browsers auf Ihrem System installiert ist). Wenn die Version "-0" entspricht ist kein Browser vorhanden, alle Nummern über "-0" bedeuten dass ein Browser installiert ist.

Die "Von" und "Bis" Felder zählen als inklusive. Das heißt das die Spalte "von 9.0.0 bis 9.2.0" die Versionen 9.0.0, 9.1.0 und 9.2.0 beinhaltet.

Im Falle der Version "5.0.0-0U" zum Beispiel würden Sie die Spalte "Kein Browser" mit der Zeile "5.0.0 bis 5.1.0" wählen. Weil Sich die Systemsoftware in dieser Reichweite befindet und kein Browser installiert ist.
{% endcapture %}

<div class="notice--info">{{ notice-1 | markdownify }}</div>

For all versions you can [Cart Update](cart-update) to a higher version in the same column to follow its instructions instead.
{: .notice--warning}

If you updated with a game cart containing the version 9.9.0 or above *(meaning that you have a version on 9.9.0 or above, but your browser is on -25 or lower, such as 10.2.0-24)*, your browser was removed and you will need to use the "No Browser" column.
{: .notice--warning}

Die Version Ihres Nintendo 3DS finden Sie rechts unten auf dem oberen Bildschirm in den Systemeinstellungen.
{: .notice--success}
