Wähle die passende Methode für deine Version aus der unteren Tabelle.
{: .notice--primary}

{% capture notice-1 %}
Die ersten beiden Spalten beziehen sich auf die letzte Nummer deiner Version (welche mit der auf dem System installierten Version des Browsers übereinstimmt). Ist die Version -0, so ist kein Browser installiert, während jede Nummer über -0 anzeigt, dass ein Browser installiert ist.

Die "Von"- und "Bis"-Felder beinhalten auch "zwischenliegende" Versionen. Das heißt, dass die Zeile "von 9.0.0 bis 9.2.0" die Versionen 9.0.0, 9.1.0 und 9.2.0 beinhaltet.

Im Falle der Version "5.0.0-0U" zum Beispiel würdest du die Spalte "Kein Browser" in der Zeile "von 5.0.0 bis 5.1.0" wählen, da sich die Systemsoftware in diesem Intervall befindet und kein Browser installiert ist.
{% endcapture %}

<div class="notice--info">{{ notice-1 | markdownify }}</div>

Für alle Versionen kannst du ein [Karten-Update](cart-update) zu einer höheren Version in der selben Spalte durchführen, damit du die Schritte von der Version stattdessen durchführen kannst.
{: .notice--warning}

Falls du mit einer Game Karte geupdated hast, die die Version 9.9.0 oder höher enthält *(d.h. du hast Version 9.9.0 oder höher, aber dein Browser ist auf -25 oder darunter, z.B. 10.2.0-24)*, wurde dein Browser entfernt. Benutze in diesem Fall die Spalte "Kein Browser".
{: .notice--warning}

Die Version Ihres Nintendo 3DS findest du rechts unten auf dem oberen Bildschirm in den Systemeinstellungen.
{: .notice--success}
