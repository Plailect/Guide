Diese Seite erklärt Ihnen den Prozess Ihre SD Karte mit F3 auf Fehler zu überprüfen.
{: .notice}

Basierend auf der Größe der SD Karte und der Geschwindigkeit Ihres Computer kann dieser Prozess mehrere Stunden dauern!
{: .notice--info}

Diese Seite ist nur für Linux User. Falls Sie kein Linux User sind, schauen Sie sich folgende Seiten an. [H2testw (windows)](h2testw-(windows)) [F3X (mac)](f3x-(mac))
{: .notice--info}

#### Was Sie brauchen

Die aktuellste Version von F3 (https://github.com/AltraMayor/f3/archive/v6.0.zip)

#### Anleitung

1. Entpacken Sie die "F3.zip"
2. Wechseln sie mit "cd" in das "F3"-Verzeichnis
3. Verwenden Sie "make" um F3 zu kompilieren
4. Stecken Sie Ihre SD Karte in den Computer
5. Mounten Sie Ihre SD Karte
6. Nun verwenden Sie "./fwrite <your sd card mount point>"
7. Warten Sie bis der Prozess abgeschlossen ist. Folgendermaßen könnte das Ergebnis aussehen:

		$ ./f3write /media/michel/6135-3363/
		Free space: 29.71 GB
		Creating file 1.h2w ... OK!
		...
		Creating file 30.h2w ... OK!
		Free space: 0.00 Byte
		Average Writing speed: 4.90 MB/s

8. Verwenden Sie "./f3read <your sd card mount point>"
9. Warten Sie bis der Prozess abgeschlossen ist. Folgendermaßen könnte das Ergebnis aussehen:

		$ ./f3read /media/michel/6135-3363/
		                  SECTORS      ok/corrupted/changed/overwritten
		Validating file 1.h2w ... 2097152/        0/      0/      0
		...
		Validating file 30.h2w ... 1491904/        0/      0/      0

		  Data OK: 29.71 GB (62309312 sectors)
		Data LOST: 0.00 Byte (0 sectors)
			       Corrupted: 0.00 Byte (0 sectors)
			Slightly changed: 0.00 Byte (0 sectors)
			     Overwritten: 0.00 Byte (0 sectors)
		Average Reading speed: 9.42 MB/s


Lautet Ihr Ergebnis "Data LOST: 0.00 Byte (0 sectors)" funktioniert Ihre SD Karte einwandfrei und sie können alle ".h2w" Dateien auf der SD Karte löschen.
{: .notice--success}

Falls es jedoch ein anderes Ergebnis anzeigen sollte ist Ihre SD Karte möglicherweise beschädigt. Sie sollten diese möglicherweise austauschen.
{: .notice--danger}
