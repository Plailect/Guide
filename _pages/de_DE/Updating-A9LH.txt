### Letzte Aktualisierung: 3 Nov 2016

Die eigentlich Installation von arm9loaderhax besteht aus "Payload Dateien", die in die NFIRM Partition des NAND Chips deines Geräts installiert werden, welcher sich auf dem Motherboard befindet. Diese Payloads werden nur sehr selten aktualisiert, und dienen nur dem einen Zweck, die "arm9loaderhax.bin" von der SD Karte zu starten, was in unserem Fall Luma3DS entspricht.
{: .notice}

Falls sie nicht wissen welche Version von arm9loaderhax Sie verwenden, installieren Sie einfach folgendermaßen die neuste Version. Das Installieren der neusten Version wenn Sie schon auf der neusten Version sind wird keine Folgen haben.
{: .notice--info}

Wenn Sie eine PIN in Luma gesetzt haben müssen sie diese temporär abschalten, um die Folgenden Schritte ausführen zu können. Sie können die PIN nach der Installation wieder aktivieren.
{: .notice--info}

Wenn Sie eine Payload verwenden, die nicht automatisch selbstständig initiiert (zum Beispiel Bootanim9) müssen Sie diese zu "arm9loaderhax_si.bin" anstelle von "arm9loaderhax.bin" umbenennen.
{: .notice--info}

The versions of data_input only refer to revisions of the `.zip` to be compatible with different installer versions, and are *not* the same thing as updating a9lh itself (the payload files). Sie haben keine praktischen Unterschiede nach dem Installationsprozess.
{: .notice--info}

Die folgenden Schritte werden außerdem die Payloads und die AES key Datenbanken aktualisieren.
{: .notice--success}

#### Anleitung

**Bei allen Vorgängen vorhandene Dateien auf der SD Karte ÜBERSCHREIBEN**

##### Teil 1 - Vorbereitung

1. Lösche die "aeskeydb.bin" im Stammverzeichnis deiner SD-Karte
2. Lösche, falls vorhanden, den Ordner "a9lh" im Stammverzeichnis deiner SD-Karte
3. Kopiere die heruntergeladene "aeskeydb.bin" in den "files9" Ordner auf deiner SD-Karte
4. Kopiere die "Hourglass9.bin" aus der "Hourglass9.zip" in den "/luma/payloads/" Ordner auf deiner SD-Karte, und benenne die "Hourglass9.bin" in "start_hourglass9.bin" um
5. Kopiere die "arm9loaderhax.bin" aus dem "SafeA9LHInstaller.zip" in den "/luma/payloads/" Ordner deiner SD-Karte
6. Benenne die "arm9loaderhax.bin" im "/luma/payloads/" Ordner in "down_safea9lhinstaller.bin" um
7. Kopiere den "a9lh" Ordner aus der "data_inputv4.zip" in das Stammverzeichnis deiner SD-Karte
8. Kopieren Sie _den kompletten Inhalt_ der "arm9loaderhax.zip" in den "a9lh" Ordner auf Ihrer SD Karte
9. Kopieren Sie die "arm9loaderhax.bin" aus der "Luma3DS.zip" in das Stammverzeichnis Ihrer SD Karte (gegebenenfalls vorhandene Dateien überschreiben)
10. Stecken sie die SD Karte zurück in Ihren Nintendo 3DS

##### Teil 2 - Update des Payloads

11. Starten Sie den 3DS während Sie das Steuerkreuz nach unten gedrückt halten
12. Drücken Sie "Select" um arm9loaderhax zu aktualisieren
13. Schalten Sie Ihren 3DS nach Abschluss des Vorgangs aus und Stecken Sie die SD Karte zurück in Ihren Computer
14. Löschen Sie den "a9lh" Ordner aus dem Stammverzeichnis Ihrer SD Karte
15. Löschen Sie die "down_safea9lhinsstaller.bin" aus dem "/Luma/payloads/" Ordner Ihrer SD Karte

##### Teil 3 - Konfiguration von Luma3DS

16. Stecken Sie Ihre SD Karte zurück in Ihren Nintendo 3DS und starten Sie diesen während Sie "Select" gedrückt halten
17. Benutzen Sie den "A-Knopf" und das Steuerkreuz um folgende Einstellungen zu aktivieren:    
  + **"Autoboot SysNAND"**
  + **"Use SysNAND FIRM if booting with R"**
  + **"Show NAND or user string in System Settings"**
3. Wenn Sie einen **New 3DS** verwenden, sollten Sie auch noch die folgenden Einstellungen festlegen:
  + **"New 3DS CPU" to "Clock+L2(x)"**
    + Das wird in vielen Spielen die Bildrate erhöhen, kann aber auch zu Instabilität in manchen Spielen führen
    + Falls manche Spiele nicht mehr richtig funktionieren sollten, können Sie diese Einstellung auch nachträglich deaktivieren
4. Drücken Sie "Start" um die Einstellungen zu Speichen und Ihren 3DS neu zu starten

##### Teil 4 - CTRNAND Luma3DS

18. Öfnnen Sie "FBI"
19. Wählen Sie "SD"
20. Navigieren Sie zu "arm9loaderhax.bin", dann drücken Sie den "A-Knopf" und wählen "Copy"
21. Kehren Sie mit dem "B-Knopf" ins Hauptmenü von FBI zurück
22. Wählen Sie "CTR NAND"
23. Wählen Sie "\<current directory>"
24. Wählen Sie "Paste", dann drücken Sie den "A-Knopf" um zu bestätigen
25. Verlassen Sie FBI mit dem "Home Button"
26. Schalten Sie Ihren 3DS aus und entfernen Sie die SD Karte
27. Boot with the SD card removed while holding (Select)
  + Booting the device at least once with the SD card removed will allow you to configure the CTRNAND based luma installation
16. Use the (A) button and the D-Pad to turn on the following:    
  + **"Show NAND or user string in System Settings"**
3. If you are using a **New 3DS**, you should *also* enable the following:
  + **"New 3DS CPU" to "Clock+L2(x)"**
    + This will increase the framerate of many games, but may cause instability in others
    + If some games do not work properly, disable this option and try again
14. Reinsert your SD card, then press Start to save and reboot!
