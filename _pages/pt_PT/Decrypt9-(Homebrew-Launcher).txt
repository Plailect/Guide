#### Necessitarás

* A versão mais recente do [Decrypt9WIP]
(https://github.com/d0k3/Decrypt9WIP/releases/latest/)
* A versão mais recente do [safehax]
(https://github.com/TiniVi/safehax/releases/latest)
* A versão mais recente do [fasthax]
(https://github.com/nedwill/fasthax/releases/latest)

#### Instruções

1. Cria uma pasta chamada 'files9' no teu Cartão SD (Apenas se ela não existir)
2. Copia o 'safehax.3dsx' para a pasta '/3ds/' presente no teu Cartão SD
3. Copia o 'fasthax.3dsx' para a pasta '/3ds/' presente no teu Cartão SD
4. Copia o 'Decrypt9WIP.bin' presente dentro de 'Decrypt9WIP.zip' para o teu Cartão SD e renomeia o.bin para 'safehaxpayload.bin'
5. Volta a inserir o Cartão SD na 3DS
6. Acede a homebrew launcher
7. Corre o fasthax
8. Depois de completo, pressiona (Start) para voltar para o homebrew launcher
  + Isto poderá levar várias tentativas
9. Inicia o fasthax
10. Se o exploit teve sucesso, estarás no Decrypt9
