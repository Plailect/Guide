Όλες οι κασέτες περιέχουν την απαιτούμενη έκδοση συστήματος για να λειτουργήσουν, επιτρέποντας χρήση της κασέτας για αναβάθμιση έκδοσης (ακόμα και χωρίς Internet).
{: .notice--info}

Τα updates από κασέτες θα αναβαθμίσουν μόνο τα βασικά στοιχεία του συστήματος όπως System Settings, Home Menu, Nintendo 3DS Sound, κλπ. Cart Updates will not deliver updates to Network features, such as System Transfer, Internet Browser, StreetPass Mii Plaza or eShop.
{: .notice--info}

Αυτό σημαίνει πως η αναβάθμιση από κασέτα ΔΕΝ θα εγκαταστήσει πρόγραμμα περιήγησης, εάν δεν έχετε ήδη ένα. Θα αναβαθμίσει <strong>μόνο</strong> την έκδοση του συστήματος <em>(για παράδειγμα, 2.1.0-0 -> 6.0.0-0)</em>. The last number *(the -0)* will **not** change.
{: .notice--warning}

**Updating with a game cart containing the version 9.9.0 or above will remove your browser!**
{: .notice--danger}

#### What you need

* A game cart that [contains the version](http://www.3dsdb.com/) specified in the chart

#### Instructions

1. Boot your 3DS into recovery mode by holding L+R+A+UP while powering on
2. Decline the update and reboot
3. Put the game in the system
4. Attempt to launch the game
5. Update when prompted
