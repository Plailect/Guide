---
title: "Get Started"
---

This page is for instructions on how to install boot9strap to a *stock* 3DS or 2DS. If you already have arm9loaderhax installed and are looking to update to boot9strap, follow [Updating to boot9strap](updating-to-boot9strap).
{: .notice--primary}

Τα χρώματα των συσκευών που απεικονίζονται μπορεί να ποικίλουν, αλλά προσέξτε την τοποθέτηση των κουμπιών και τα χαρακτηριστικά της κάθε μίας, έτσι ώστε να είστε σίγουροι πως επιλέξατε τη σωστή συσκευή.
{: .notice}

Πατήστε στην εικόνα της συσκευής σας έτσι ώστε να μεταφερθείτε στην κατάλληλη σελίδα.
{: .notice--primary}

Different device models, versions, and regions will require different steps to achieve the end goal of boot9strap Custom Firmware. Αυτή η σελίδα θα σας βοηθήσει να βρείτε από που να αρχίσετε για τη συσκευή σας.
{: .notice--info}

Εάν έχετε ήδη χακάρει το 3DS σας πριν και έχετε ήδη μια εγκατάσταση CFW βασισμένη σε emuNAND, απλώς ακολουθήστε τις οδηγίες ενώ βρίσκεστε στη SysNAND. Your data will be transfered from EmuNAND to your new B9S install at the end.
{: .notice--info}

Note that screenshots in this guide can be enlarged to show more context by clicking on them.
{: .notice--info}

Before starting, Windows users should enable the option to show file extensions using [File Extensions (Windows)](file-extensions-(windows))!
{: .notice--info}

Πριν αρχίσετε, συνιστάται να ελέγξετε την κάρτα SD για σφάλματα χρησιμοποιώντας το [H2testw (Windows)](h2testw-(windows)), το [F3 (Linux)](f3-(linux)), ή το [F3X (Mac)](f3x-(mac))!
{: .notice--warning}

Note that the new [Installing boot9strap (Hardmod)](installing-boot9strap-(hardmod)) method works on *all devices, regions, and versions*! This includes CHN and TWN region devices that were previously unhackable! Currently, for the sake of having a cleaned up chart, it is only listed for versions where it is the only option. *This does not mean that it only works on these versions.*
{: .notice--success}
