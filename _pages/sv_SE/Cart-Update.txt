Alla spelkassetter innehåller en kopia av systemversionen som krävs för att spela det. Detta tillåter alla som använder en lägre systemversion att använda kassetten för att uppdatera till den versionen, utan att behöva en internetanslutning.
{: .notice--info}

Kassettuppdateringar kommer enbart att uppdatera kärnfunktioner, som Systeminställningar, Hemmenyn, Nintendo 3DS Sound osv. Cart Updates will not deliver updates to Network features, such as System Transfer, Internet Browser, StreetPass Mii Plaza or eShop.
{: .notice--info}

This means that a cart update will NOT install a browser if you do not already have one. This will **only** update the actual system version *(for example, 2.1.0-0 -> 6.0.0-0)*. The last number *(the -0)* will **not** change.
{: .notice--warning}

**Updating with a game cart containing the version 9.9.0 or above will remove your browser!**
{: .notice--danger}

#### What you need

* A game cart that [contains the version](http://www.3dsdb.com/) specified in the chart

#### Instructions

1. Boot your 3DS into recovery mode by holding L+R+A+UP while powering on
2. Decline the update and reboot
3. Put the game in the system
4. Attempt to launch the game
5. Update when prompted
