### Lettura necessaria

All game cartridges contain a copy of the required system version to play it, allowing anyone on a lower version to use the cartridge to update to that version (even without an internet connection).

{% capture notice-1 %}

Note that cartridge updates will only deliver updates to core features, such as the System Settings, Home Menu, etc. cartridge updates will not deliver updates to Nintendo 3DS Sound and Network features, such as System Transfer, Internet Browser, StreetPass Mii Plaza, or eShop.

This means that using a cartridge update from a version containing an older Nintendo 3DS Sound version *(<3.0.0)* to one that introduced a newer Nintendo 3DS Sound version will break [Soundhax](homebrew-launcher-(soundhax))! Dovrai utilizzare un [metodo alternativo](homebrew-launcher-(alternatives)) per avviare l'Homebrew Launcher!

This also means that a cartridge update will NOT install a browser if you do not already have one. Aggiornerà **solamente** la versione di sistema *(ad esempio, 2.1.0-0 -> 6.0.0-0)*. L'ultimo numero *(lo -0)* *non* cambierà.
{% endcapture %}

<div class="notice--warning">{{ notice-1 | markdownify }}</div>

**Updating with a game cartridge containing the version 9.9.0 or above will remove your browser until a standard online update is performed!**
{: .notice--danger}

### Cosa serve

* A game cartridge that [contains the version](http://www.3dsdb.com/) you wish to update to

### Istruzioni

1. Avvia il 3DS in modalità di ripristino tenendo premuto (L) + (R) + (A) + (Su) durante l'accensione
1. Se richiesto, rifiuta l'aggiornamento
1. Riavvia la console
1. Inserisci la scheda di gioco nella console nel caso non fosse già stata inserita
1. Prova ad avviare il gioco
1. Aggiorna quando richiesto
