# # # Ultimo aggiornamento: 3 novembre 2016

L'installazione reale di arm9loaderhax stesso consiste in payload che risiedono nelle partizioni NFIRM della NAND del tuo dispositivo, che è saldata a sua volta alla scheda madre. Questi payload sono aggiornati raramente e hanno il solo scopo di lanciare "arm9loaderhax.bin" dalla scheda SD, che è, nel nostro caso, Luma3DS.
{: .notice}

Se non conosci quale versione dell'arm9loaderhax è in esecuzione, allora installa l'ultima versione con questi passaggi. Installare l'ultima versione mentre sull'ultima non avrà già nessun effetto.
{: .notice--info}

Se si dispone di un PIN impostato su Luma, è necessario temporaneamente disattivare ed eseguire questi passaggi. È possibile riattivare il PIN dopo l'aggiornamento.
{: .notice--info}

Se si utilizza un payload che non inizializza lo schermo (come Bootanim9), sarà necessario rinominarlo in 'arm9loaderhax_si.bin' invece di 'arm9loaderhax.bin'
{: .notice--info}

Solo le versioni di data_input si riferiscono a revisioni della '. zip' per essere compatibile con le differenti versioni dell' installer e * non * sono la stessa cosa come l'aggiornamento stesso di a9lh (i file di payload). Non fanno alcuna differenza pratica oltre il processo di installazione.
{: .notice--info}

Questa procedura aggiornerà anche i vostri vari payload e il database delle chiavi AES.
{: .notice--success}

#### Istruzioni

* * Per le istruzioni, SOVRASCRIVERE i file esistenti sulla vostra SD card * *

##### Sezione I - Preparazione

1. Elimina qualsiasi 'aeskeydb.bin' esistente dalla root della vostra scheda SD
2. Se presente, elimina la cartella 'a9lh' dalla root della tua scheda SD
3. Copia 'aeskeydb.bin' nella cartella '/files9/' sulla tua scheda SD
4. Copia 'Hourglass.bin' da Hourglass9 'zip' alla cartella '/luma/payloads/' sulla tua scheda SD e rinomina 'Hourglass9.bin' in 'start_Hourglass9.bin'
5. Copia 'arm9loaderhax.bin' da SafeA9LHInstaller ' .zip' alla cartella '/luma/payloads/' sulla tua scheda SD
6. Rinomina 'arm9loaderhax.bin ' nella cartella ' /luma/payloads/' in ' down_safea9lhinstaller.bin '
7. Copia la cartella `a9lh` presente nell'archivio data_input `.zip` nella root della tua scheda SD**
8. Copia _il contenuto_ di arm9loaderhax ' .zip ' nella cartella `a9lh` della tua scheda SD
9. Copia ' arm9loaderhax.bin ' da Luma3DS ' .zip' alla root della tua scheda SD, sovrascrivendo i file esistenti
10. Reinserisci la tua scheda SD nel tuo Nintendo 3DS

##### Sezione II - Aggiornamento Payload

1. Accendi la console tenendo premuto la freccetta Giù
2. Premi (Select) per aggiornare arm9loaderhax
3. Spegni la console e inserisci la scheda SD nel tuo computer
4. Elimina la cartella 'a9lh' dalla root della tua scheda SD
5. Cancella il file `down_safea9lhinstaller.bin` dalla cartella `/luma/payloads/`

##### Sezione III - Configurare Luma3DS

1. Reinserisci la scheda SD nel 3DS e avvia la console tenendo premuto (Select)
2. Usa il pulsante (A) e il D-Pad per abilitare le seguenti opzioni:    
  + **"Autoboot SysNAND"**
  + **"Use SysNAND FIRM if booting with R"**
  + **"Show NAND or user string in System Settings"**
3. Se stai utilizzando un **New 3DS**, dovresti selezionare *anche*:
  + **"New 3DS CPU" su "Clock+L2(x)"**
    + Questa opzione aumenterà il framerate in molti giochi, ma può rendere instabili altri
    + Se alcuni giochi non dovessero più funzionare correttamente, disabilita questa opzione
4. Premi (Start) per salvare e riavviare il 3DS

##### Sezione IV - CTRNAND Luma3DS

1. Apri FBI
2. Seleziona "SD"
3. Seleziona il file `arm9loaderhax.bin`, quindi premi (A) e seleziona l'opzione "Copy"
4. Ritorna al menu principale di FBI premendo (B)
5. Seleziona "CTR NAND"
6. Seleziona "\<current directory>"
7. Seleziona l'opzione "Paste", quindi premi (A) per confermare la scelta
8. Esci premendo il pulsante Home
9. Spegni il tuo 3DS e rimuovi la scheda SD
10. Riavvia la console con la scheda SD rimossa e tenero premuto il pulsante (Select)
  + Il riavvio della console con la scheda SD rimossa permette la configurazione dell'installazione CTRNAND di Luma
11. Usa il pulsante (A) e il D-Pad per abilitare le seguenti opzioni:    
  + **"Show NAND or user string in System Settings"**
12. Se stai utilizzando un **New 3DS**, dovresti selezionare *anche*:
  + **"New 3DS CPU" su "Clock+L2(x)"**
    + Questa opzione aumenterà il framerate in molti giochi, ma può rendere instabili altri
    + Se alcuni giochi non dovessero più funzionare correttamente, disabilita questa opzione
13. Reinserisci la scheda SD, quindi premi (Start) per salvare e riavviare la console!
