Questa è una sezione aggiuntiva per spostare il contenuto di una EmuNAND al tuo nuovo CFW della SysNAND, e poi rimuovere la vecchia partizione EmuNAND. Nota che i termini EmuNAND e RedNAND si riferiscono a due implementazioni leggermente diverse [dello stesso concetto](http://3dbrew.org/wiki/NAND_Redirection).
{: .notice--info}

Perderete i vostri salvataggi GBA VC e DSiWare!
{: .notice--danger}

**È necessario aver già installato arm9loaderhax e Luma3DS per poter procedere.**
{: .notice--danger}

#### Quello che ti serve

*Una EmuNAND esistente
*L'ultima release di [GodMode9](https://github.com/d0k3/GodMode9/releases/latest)

#### Istruzioni

1. Copia `GodMode9.bin` da GodMode9 `.zip` nella cartella `/luma/payloads/` sulla tua scheda SD e rinomina 
 `GodMode9.bin` ora nella cartella `/luma/payloads/` in `up_GodMode9.bin`
2. Reinserisci la tua scheda SD nel tuo Nintendo 3DS
3. Apri Hourglass9 dall'arm9loaderhax tenendo premuto (Start) all'avvio
4. Vai a "EmuNAND Backup/Restore",e seleziona l'opzione "EmuNAND Backup" per fare un backup della tua EmuNAND chiamato `NANDmin_emu.bin`
5. Premi (B) per tornare al menu principale
6. Andare a "SysNAND Backup/Restore", quindi selezionare l'opzione "Ripristino SysNAND (tenere a9lh)" per ripristinare il tuo SysNAND da 'NANDmin_emu.bin'
7. Premere (Select) per estrarre la scheda SD
8. Inserisci la scheda SD nel computer, quindi copia i file "NANDmin_emu.bin" e "NANDmin_emu.bin.sha" dalla cartella `/files9/` presente nella scheda SD ad una posizione sicura
  + Crea delle copie e mettile in più locazioni
  + Questo backup ti salverà da un brick nel caso succeda qualcosa in futuro
  + **Il file di backup dovrà avere l'esatta dimensione riportata su [questa](nand-size) pagina; se così non fosse dovresti cancellarlo e crearne uno nuovo!**
9. Cancella `NANDmin_emu.bin` dalla cartella `/files9/` che si trova sulla tua scheda SD dopo averne fatta una copia
10. **Fai un backup di ogni file sulla tua scheda SD e conservalo in una cartella sul tuo computer, tutti i file verranno cancellati nel prossimo passaggio**
11. Reinserisci la tua scheda SD nel tuo Nintendo 3DS
12. Premi (Start) per riavviare la console e contemporaneamente tieni premuto (Su) per aprire GodMode9
13. Premi il tasto (Home) per far apparire il menu delle azioni
14. Seleziona "SD format menu"
15. Premi il pulsante (A) per confermare
16. Seleziona "nessuna EmuNAND"
17. Seleziona "Auto"
18. Inserisci la combinazione di tasti che appare a schermo
19. Tieni premuto (R) e premi (B) allo stesso tempo per rimuovere la tua scheda SD
20. Metti la tua scheda SD nel tuo computer, poi copiaci sopra i file precedentemente salvati
  +Assicurati di sostituire il file `arm9loaderhax.bin` sulla tua scheda SD con quello preso dal tuo backup
21. Reinserisci la tua scheda SD nel tuo Nintendo 3DS
22. Premi (A) per reinserire la scheda SD
23. Premi il pulsante (Start) per riavviare la console!
24. + Se ottieni una schermata nera, [segui questa guida per capire il problema](troubleshooting#ts_sys_down)

Ritorna a [Installare arm9loaderhax](installing-arm9loaderhax).
