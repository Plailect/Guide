Questa pagina ti guiderà nel processo di ricerca problema della tua scheda SD usando F3X.
{: .notice}

A seconda delle dimensioni della tua scheda SD e della velocità del tuo computer, questo procedimento può durare diverse ore!
{: .notice--info}

Questa pagina è solo per gli utenti Linux. Se non state usando Linux, controllate [H2testw (windows)](h2testw-(windows)) o [F3X (mac)](f3x-(mac)).
{: .notice--info}

#### Quello che ti serve

* L'ultima versione di [F3] (https://github.com/AltraMayor/f3/archive/v6.0.zip)

#### Istruzioni

1. Estrai il file F3 `.zip`
2. Utilizza il comando `cd` per aprire la cartella di f3
3. Lancia 'make' per compilare F3
4. Inserisci la tua scheda SD nel tuo computer
5. Inserisci la scheda SD
6. Esegui '. / f3read <Mount point della tua scheda Sd>'
7. Attendi che il processo sia completato. Vedi sotto per un esempio di output.

		$ ./f3write /media/michel/6135-3363/
		Spazio libero: 29.71 GB
		Creating file 1.h2w ... OK!
		...
		Creating file 30.h2w ... OK!
		Spazio libero: 0.00 Byte
		Velocità di scrittura media: 4.90 MB/s

8. Esegui '. / f3read <your sd card mount point>'
9. Attendi che il processo sia completato. Vedi sotto per un esempio di output.

		$ ./f3read /media/michel/6135-3363/
		                  SECTORS      ok/corrupted/changed/overwritten
		Validating file 1.h2w ... 2097152/        0/      0/      0
		...
		Validating file 30.h2w ... 1491904/        0/      0/      0

		  Data OK: 29.71 GB (62309312 sectors)
		Data LOST: 0.00 Byte (0 sectors)
			       Corrupted: 0.00 Byte (0 sectors)
			Slightly changed: 0.00 Byte (0 sectors)
			     Overwritten: 0.00 Byte (0 sectors)
		Average Reading speed: 9.42 MB/s


Se il test mostra il risultato ' dati PERSI: 0.00 Byte (0 settori)' la scheda SD è buona ed è possibile eliminare tutti i file '.h2w' dalla scheda SD
{: .notice--success}

Se il test mostra qualsiasi altro risultato, la tua scheda SD potrebbe essere corrotta o danneggiata e potresti doverla sostituire!
{: .notice--danger}
