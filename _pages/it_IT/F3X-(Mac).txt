Questa pagina ti guiderà nel processo di ricerca problema della tua scheda SD usando F3X.
{: .notice}

A seconda delle dimensioni della tua scheda SD e della velocità del tuo computer, questo procedimento può durare diverse ore!
{: .notice--info}

Questa pagina è solo per utenti che usano un Mac. Se non stai usando un mac, controlla le pagine [H2testw (windows)](h2testw-(windows)) o [F3 (Linux)](f3-(linux)).
{: .notice--info}

#### Di cosa hai bisogno

* L'ultima release di [F3X](https://github.com/insidegui/F3X/releases)

#### Istruzioni

1. Estrai il file F3X `.zip`
2. Inserisci la tua scheda SD nel tuo computer
3. Avvia l'applicazione F3X
4. Seleziona la tua scheda SD
5. Premi "Start Test"
6. Attendi che il processo sia completato.

Se il test mostra `Success! come risultato La tua scheda funziona correttamente! la tua scheda SD funziona bene e puoi cancellare tutti i file `.h2w` dalla tua scheda SD
{: .notice--success}

Se il test mostra qualsiasi altro risultato, la tua scheda SD potrebbe essere corrotta o danneggiata e potresti doverla sostituire!
{: .notice--danger}

Ritorna a [Cominciamo](get-started)
{: .notice--primary}
