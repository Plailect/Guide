Select th' appropriate page fer ye version from th' chart below.
{: .notice--primary}

{% capture notice-1 %}
The last two columns of the chart refer to the last number of your version (which corresponds to the browser version installed to the system). If th' version be -0 then ye do not have a browser, while any number above -0 indicates a browser be installed.
<br><br>
Th' "from" 'n "to" fields be inclusive. 'Tis means that, fer example, th' "from 9.0.0 to 9.2.0" row be includin' 9.0.0, 9.1.0, 'n 9.2.0.
<br><br>
In th' case 'o "5.0.0-0U" fer example, ye would follow th' "No Browser" column 'n 5.0.0 to 5.1.0 row because th' system be on th' a system version in that range 'n has no browser installed.
{% endcapture %}

<div class="notice--info">{{ notice-1 | markdownify }}</div>

Fer all versions ye can also [Cart Update](cart-update) to a higher version in th' same column to follow its instructions.
{: .notice--info}

{% capture notice-1 %}

Note that Cart Updates will only deliver updates to core features, such as the System Settings, Home Menu, etc. Cart Updates gunna not be deliverin' updates to Nintendo 3DS Sound 'n Network features, such as System Transfer, Internet Browser, StreetPass Mii Plaza, or eShop.
<br><br>
'Tis be meanin' that usin' a Cart Update from a version containin' an older Nintendo 3DS Sound version *(<7.0.0 fer Old 3DSes in th' EUR, JPN, KOR, 'n USA regions)* to one that introduced a newer Nintendo 3DS Sound version gunna be breakin' [Soundhax](homebrew-launcher-(soundhax))! Ye gunna be needin' an [alternate method](homebrew-launcher-(alternatives)) 'o enterin' th' Homebrew Launcher!
{% endcapture %}

<div class="notice--warning">{{ notice-1 | markdownify }}</div>

If ye updated wit' a game cart containin' th' version 9.9.0 or above *(meanin' that ye have a version on 9.9.0 or above, but ye browser be on -25 or lower, such as 10.2.0-24)*, ye browser was removed 'n ye gunna need to be usin' th' "No Browser" column.
{: .notice--warning}

Ye device version can be found at th' bottom right 'o th' top screen 'o th' System Settin's.
{: .notice--success}
