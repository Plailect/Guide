### Lectura requerida

All game cartridges contain a copy of the required system version to play it, allowing anyone on a lower version to use the cartridge to update to that version (even without an internet connection).

{% capture notice-1 %}

Note that cartridge updates will only deliver updates to core features, such as the System Settings, Home Menu, etc. cartridge updates will not deliver updates to Nintendo 3DS Sound and Network features, such as System Transfer, Internet Browser, StreetPass Mii Plaza, or eShop.

This means that using a cartridge update from a version containing an older Nintendo 3DS Sound version *(<3.0.0)* to one that introduced a newer Nintendo 3DS Sound version will break [Soundhax](homebrew-launcher-(soundhax))! ¡Necesitarás un [método alternativo](homebrew-launcher-(alternatives)) para entrar al Homebrew Launcher!

This also means that a cartridge update will NOT install a browser if you do not already have one. Esto **sólo** actualizará tu versión del sistema *(por ejemplo, 2.1.0-0 -> 6.0.0-0)*. El último número *(el -0)* **no** cambiará.
{% endcapture %}

<div class="notice--warning">{{ notice-1 | markdownify }}</div>

**Updating with a game cartridge containing the version 9.9.0 or above will remove your browser until a standard online update is performed!**
{: .notice--danger}

### Qué necesitas

* A game cartridge that [contains the version](http://www.3dsdb.com/) you wish to update to

### Instrucciones

1. Inicia tu consola en modo de recuperación pulsando (L) + (R) + (A) + (Arriba) mientras presionas el botón de encendido
1. Si se solicita, rechaza la actualización
1. Reinicia tu consola
1. Pon el juego en tu consola si aún no lo está
1. Intenta iniciar el juego
1. Actualiza cuando te pregunte
